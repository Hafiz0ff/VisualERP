# VisualERP Roadmap

## v0.1 - Documentation Foundation

Deliverables:

- project context;
- product specification;
- architecture baseline;
- engineering rules;
- AI-agent instructions;
- roadmap and ADR.

## v0.2 - Domain Model

Deliverables:

- refined entity model;
- invariants and state transitions;
- module ownership rules;
- terminology profile strategy.

## v0.3 - Backend MVP

Deliverables:

- first backend structure;
- core inventory and production flows;
- initial auth and permissions baseline.

## v0.4 - Frontend Integration

Deliverables:

- integration of documented backend contracts with the existing prototype direction;
- first operational screens wired to real APIs.

## v0.5 - Reports

Deliverables:

- stock reports;
- production reports;
- shipment and write-off reports;
- initial dashboard summary views.

## v0.6 - First Pilot

Deliverables:

- first real business pilot;
- migration learning from Excel data;
- feedback-driven corrections.

## v1.0 - Production-Ready MVP

Deliverables:

- stable core modules;
- role and audit maturity;
- operational hardening;
- deployment and backup baseline.

## Future Industry Profiles

After the initial pilot, the product should expand profile support for other small manufacturing industries without rewriting the core domain.
