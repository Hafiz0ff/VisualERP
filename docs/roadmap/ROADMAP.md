# VisualERP Roadmap

## v0.1 - Documentation Foundation

Deliverables:

- project context;
- product specification;
- architecture baseline;
- engineering rules;
- AI-agent instructions;
- roadmap and ADR.

## v0.2 - Domain Model

Deliverables:

- refined entity model;
- invariants and state transitions;
- module ownership rules;
- terminology profile strategy.

## v0.3 - Backend MVP (Complete)

Deliverables:
- first backend structure;
- core inventory and production flows;
- initial auth and permissions baseline.

## v0.4 - Frontend Integration (Complete)

Deliverables:
- integration of documented backend contracts with the existing prototype direction;
- first operational screens wired to real APIs.

## v0.5 - Reports (Complete)

Deliverables:
- stock reports;
- production reports;
- shipment and write-off reports;
- initial dashboard summary views.

## v0.6 - First Pilot (Complete - v0.9.0-beta)

Deliverables:
- first real business pilot release packaging (Docker Compose orchestration with Nginx reverse proxy);
- programmatic E2E workflow script validation;
- operational guides for deployment, migration, backup, and restore;
- pilot checklist and limitations documentation.

## v1.0 - Production-Ready MVP (Next Phase)

Deliverables:
- stable core modules with full JWT-based session auth;
- role-based permission checks (RBAC) enforced on host endpoints;
- frontend routing guards and permission-scoped capability toggles;
- feedback-driven corrections from first pilot customer.

## Future Industry Profiles

After the initial pilot, the product should expand profile support for other small manufacturing industries without rewriting the core domain.
